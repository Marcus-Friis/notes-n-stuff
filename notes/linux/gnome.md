# Gnome

Install [Gnome](https://wiki.archlinux.org/title/GNOME) with
```
sudo pacman -S gnome
```

Accept all the default installation stuff, and boot into it from the [display manager](https://wiki.archlinux.org/title/Display_manager). 

## Plugins

You can install [Gnome plugins](https://extensions.gnome.org/) using their website and the web extension. 
* [Dash-to-dock](https://micheleg.github.io/dash-to-dock/)
