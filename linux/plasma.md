# KDE Plasma

>[KDE](https://en.wikipedia.org/wiki/KDE "wikipedia:KDE") is a software project currently comprising a [desktop environment](https://wiki.archlinux.org/title/Desktop_environment "Desktop environment") known as [Plasma](https://en.wikipedia.org/wiki/KDE_Plasma "wikipedia:KDE Plasma"), a collection of libraries and frameworks (KDE Frameworks) and several applications (KDE Applications) as well. - [KDE Arch docs](https://wiki.archlinux.org/title/KDE)

Get inspired by [Tony's Tutorial](https://www.youtube.com/watch?v=XAIORfutPyk) for customizing KDE Plasma.
## Visual theme

**To apply get a good default theme do**
1) Pick *Global Theme* `WhiteSur-dark` - apply with desktop and window layout
2) Change *Icons* to `Tela nord`

### Konsole

**Configure Konsole**
Open `Menu` -> `Settings` -> `Configure Konsole` (`CTRL+SHIFT+,`).
* Disable `Remember window size` 
* Create new profile and make it default
	* Alter font and size
	* Add opacity to window (edit  color scheme -> 10% background color transparency)
	* Scrolling -> Hidden to hide scroll bar (only visual)

**Remove Toolbar Menus**
Right click the toolbar and disable `Lock Toolbar Positions` followed by  disabling `Main Toolbar` and `Session Toolbar`

## Keybindings

Go to *System Settings* -> *Keyboard* -> *Shortcuts*
* *Krunner* - Bind `SUPER+D`
* *Konsole* - Bind `SUPER+RETURN` or `SUPER+T`
* *Browser* - Bind Firefox to `SUPER+B`
* Maximize window - *Window Management* -> *Maxmize Window* bind to `SUPER+SHIFT+F`
* Close Window - *Window Management* -> *Close Window* bind to `SUPER+Q`
